ESP32 Power Converter Efficiency Monitor

Designed and built an embedded monitoring system for a DC-DC buck converter using an ESP32-S3 running FreeRTOS. Implemented a multitasking firmware architecture with independent tasks for sensor sampling, data logging, WiFi/MQTT cloud reporting, and CAN bus broadcasting, coordinated through FreeRTOS queues. Measured real-time voltage, current, and temperature using a shunt-resistor current sensing circuit and a DS18B20 probe, computing live power conversion efficiency and comparing results against the converter's datasheet specifications.

ESP32 Industrial Condition Monitoring Node

Developed an embedded condition-monitoring system using an ESP32 running Zephyr RTOS. The device collects temperature and vibration measurements, processes the sensor data locally, and detects abnormal operating conditions using RMS and threshold-based analysis. The firmware uses separate Zephyr threads for sensor acquisition, signal processing, data storage, and network communication, with message queues used for safe data transfer between components.

Implemented secure WiFi communication and MQTT-based telemetry for publishing live measurements, alarms, system uptime, and device status. Added persistent configuration storage, automatic network reconnection, watchdog-based fault recovery, structured logging, and planned support for signed firmware updates using MCUboot and MCUmgr.
